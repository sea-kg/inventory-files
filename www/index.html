About

The program is freeware. All sources published on github.com. A program will be help full for watching changes your files. Use program you can scan files and search by filenames, type, size. Also this program will be detection type of file by path and extensions. You can write description for files.
  • Perform an inventory of the files on your computer
  • May be reports
Used Technologies:
  • C++
  • Qt5 Framework
  • Database SQLite

Sources

You can donwload source code from here: GitHub.com

Or Build & Run (for Ubuntu or Debian): 
$ sudo apt-get install git-core
$ sudo apt-get install qt-sdk
$ cd ~
$ git clone https://github.com/sea-kg/inventory-files.git inventory-files
$ cd inventory-files
$ qmake
$ make
$ ./inventory-files.bin
